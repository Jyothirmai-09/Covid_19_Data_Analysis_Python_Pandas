# Covid_19_Data_Analysis_Python_Pandas

## 📌 Project Overview

This project focuses on analyzing global Covid-19 data using **Python, Pandas, Matplotlib, and Seaborn**. The objective was to perform data cleaning, preprocessing, exploratory data analysis (EDA), and statistical analysis to uncover trends in confirmed, recovered, and death cases across different regions.

By leveraging Python's data analysis capabilities, this project demonstrates how real-world healthcare datasets can be transformed into meaningful insights through filtering, aggregation, sorting, and visualization techniques.

---

## 🎯 Objectives

* Analyze Covid-19 case statistics across different regions.
* Perform data cleaning and preprocessing.
* Handle and visualize missing values.
* Compare confirmed, recovered, and death cases region-wise.
* Identify regions with the highest and lowest Covid-19 impact.
* Apply filtering, sorting, and aggregation techniques for deeper analysis.

---

## 📂 Dataset Information

The dataset contains Covid-19 statistics reported from various regions and countries.

### Features

* Region / Country
* Confirmed Cases
* Death Cases
* Recovered Cases
* Observation Date

Each record represents Covid-19 statistics reported for a specific region.
